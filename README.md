# Maplas — Country Guessing Game
 
Maplas is a browser-based country trivia game built with **Flask**. The idea behind the project was simple: collect as much publicly available information as possible about every country in the world such as capitals, flags, anthems, borders, flag colours, country outlines, alcohol consumption data, and more we combined it all into a fun and replayable guessing game that tests your geography and general knowledge.
 
Each round, a mystery country is chosen at random. You are given clues one by one and have to figure out which country it is before you run out of guesses. The clues are pulled from the real-world data scraped during setup, meaning every fact you see in the game is genuine public information. The more knowledge you have on all the countries, the better your chances.
 
The project was built in two parts: a **scraping pipeline** that gathers and cleans all country data from the web, and a **Flask web app** that uses that data to power the game in your browser.

##  Getting Started
 
### 1. Clone the repository
 
```bash
git clone https://github.com/YOUR-USERNAME/Maplas.git
cd Maplas
```
 
### 2. (Recommended) Create a virtual environment
 
```bash
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows
```
 
### 3. Install dependencies
 
```bash
pip install -r requirements.txt
```
 
### 4. Run the Flask app
 
```bash
python app.py
```
 
Then open your browser and go to: [http://localhost:5000](http://localhost:5000)

## First-Time Data Setup
 
The game requires a local dataset of country information generated by the scraping pipeline. **If this is your first time running the app**, it will automatically redirect you to the `/setup` page.
 
From there, click **"⬇ Scrape data now"** and the app will run all scraper scripts in sequence, with a live progress bar:
 
| Step | Script | What it collects |
|------|--------|-----------------|
| 1 | `country_collection.py` | Country list from Wikipedia |
| 2 | `scrape_country_info.py` | Capitals, flags, anthems |
| 3 | `Alcohol_consumption_ranked_scraper.py` | Alcohol consumption data |
| 4 | `Bordering_Countries_Scraper.py` | Borders & neighbouring countries |
| 5 | `Country_flag_colour_scraper.py` | Flag colour data |
| 6 | `country_outline_scraper.py` | Country outline images |
| 7 | `cleaning.py` | Cleans and finalises the dataset |
 
>  This process may take several minutes. Progress is shown live on the setup page.
 
Once all steps show ✓, click **"▶ Play now"** to start the game. You can also re-scrape at any time to refresh the data.
 
---
 
## How to Play
 
1. Each round a **mystery country** is selected at random.
2. You receive an initial clue and have to guess which country it is.
3. With each wrong guess, a **new clue** is revealed (e.g. flag colours, outline, anthem, borders).
4. You have a limited number of guesses — try to keep your **streak** alive!
